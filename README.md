# 😄smile-log

### 1. INTRODUCTION

스마일게이트 STOVE DEV CAMP 2기 사전 과제 전형으로 개발하게 된 블로그 smile-log입니다.

매일, 즐거운 개발 기록을 남기고 싶다는 생각이 들어 작명하였습니다.

Smilegate stove를 상징하는 두 가지 색상을 테마로 사용하였고 velog를 참고하여 개발하였습니다.
